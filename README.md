# 🎱 CueSphere



## CueSphere is a simple poll application:



## Installation

### 💾 After cloning the application run the following command

```terminal
$ npm install
$ npm run dev
```

### 🧪 For testing the application run the following command

```terminal
$ npm run test
```

## ⚙️ Technologies Used

- Next.js
- Rect.js
- Tailwind CSS
- Jest

